Font

All displayed text uses the same u8g2 font. u8g2 fonts are divided into groups, where each group has one or more fonts and each font has one or more character sets. The u8g2 font groups page links to a page for each group with the details for every group. Each details page shows the available character sets for each font in the group, along with images of the character set.

To configure the u8g2 font, change the default value of the preprocessor macro #define FONT u8g2_font_<font>_<set> to the desired font.

Typical character sets include:

  • tf: full ASCII character set plus accented characters
  • tr: full ASCII character set
  • tn: ASCII numeric characters
  • tu: ASCII character set without lowercase characters

For example, to select the nokiafc22 font with the tf character set:

#define FONT u8g2_font_nokiafc22_tf
